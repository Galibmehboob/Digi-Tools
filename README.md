Digitool is a simple and responsive web application built with React. It is designed to manage a basic cart system where users can add and remove items, view real-time total price updates, and complete a checkout process with proper validation.

Features

--> Add items to cart
--> Remove items from cart
--> Automatic total price calculation
--> Disable checkout when cart is empty
--> Clear cart after successful purchase
--> Fully responsive design for all devices
--> Clean and simple user interface

Tools and Technologies Used

--> React.js
--> JavaScript
--> Tailwind CSS
--> React Toastify
